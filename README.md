## desc 
build assets outside the node_modules folder


## usage

```bash
    # path of wordpress-assets-builder
    npm install
    npm link

    # path of wordpress-theme, containing assets/theme.js 
    # watch mode
    wp dev
    # product mode
    wp pro
```
